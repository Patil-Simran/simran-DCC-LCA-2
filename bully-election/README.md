# Task 1 — Bully election (Java)

## Steps

1. Install **JDK** → check: `javac -version` and `java -version`.
2. Open terminal in folder `bully-election`:
   ```bash
   javac BullyElection.java
   java BullyElection
   ```
3. At the prompt type **`demo`** → Enter (fails coordinator, runs election, shows new coordinator).
4. Or use **`1`** (fail + elect), **`3`** (state), **`q`** (quit).
5. Save terminal output or screenshot → put in `screenshots/`.

**Files:** `BullyElection.java`
